🌙 DreamOS Design System

«Design Philosophy: Simple. Modern. Intelligent. Lightweight.»

DreamOS is designed to be visually beautiful, fast, and intuitive while staying lightweight. Every element should have a purpose and help users achieve their goals.

---

Design Principles

Simplicity

Interfaces should be clean and uncluttered.

Consistency

Buttons, icons, menus, and layouts should behave consistently throughout the system.

Performance

Animations and effects should be smooth without sacrificing speed or battery life.

Accessibility

DreamOS should be easy to use for everyone, with support for keyboard navigation, screen readers, and customizable text sizes.

Personalization

Users should be able to customize themes, wallpapers, layouts, widgets, and accent colors.

---

Design Language

DreamOS follows a modern design language with:

- Rounded corners
- Smooth animations
- Soft shadows
- Clean typography
- Minimal visual clutter
- Responsive layouts

---

Color Palette

Primary Colors

- Midnight Blue — Main system color
- Royal Purple — Brand identity
- Cyan — AI and interactive elements
- White — Light mode
- Black / Dark Gray — Dark mode

Status Colors

- Green — Success
- Yellow — Warning
- Red — Error
- Blue — Information

---

Typography

Recommended font characteristics:

- Modern
- Highly readable
- Optimized for small and large screens
- Consistent across desktop and mobile

Different font sizes should be used for:

- Display titles
- Headings
- Body text
- Captions
- Code

---

Icons

DreamOS icons should be:

- Minimal
- Rounded
- Easy to recognize
- Consistent in style
- Scalable for different screen sizes

---

Logo

The DreamOS logo consists of:

- Crescent Moon 🌙
- Star ⭐
- Hidden "D" shape
- Optional circuit-inspired details representing AI
- Optional shield representing security

The logo should remain recognizable at all sizes.
