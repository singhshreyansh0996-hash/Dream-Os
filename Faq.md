🌙 DreamOS Frequently Asked Questions

Welcome to the DreamOS FAQ! This document answers common questions about the project.

---

General Questions

What is DreamOS?

DreamOS is a long-term project that aims to create a lightweight, secure, AI-powered operating system for both desktop and mobile devices.

---

What makes DreamOS different?

DreamOS is built around the idea of helping users achieve their goals instead of simply providing access to applications.

The long-term vision includes:

- AI integration
- Strong security
- Modern design
- Cross-device experience
- Developer-friendly tools

---

Is DreamOS open source?

The licensing model is still under discussion and may change as the project evolves.

---

Which devices will DreamOS support?

The long-term goal is to support:

- Desktop computers
- Laptops
- Tablets
- Smartphones

Security

What is DreamShield?

DreamShield is the planned built-in security platform for DreamOS.

Its goal is to help protect users through features such as malware detection, firewall capabilities, permission management, and recovery tools.

---

What is Emergency Mode?

Emergency Mode is a planned recovery environment designed to help users access essential tools if they believe their system has been compromised.

---
