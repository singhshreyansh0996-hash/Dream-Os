Code of Conduct

DreamOS Community Guidelines

Welcome to the DreamOS community!

DreamOS aims to build a modern, lightweight, secure, AI-powered operating system. To achieve this, we want to maintain a respectful, inclusive, and collaborative environment.

---

Our Values

- Respect everyone.
- Be kind and professional.
- Welcome new members.
- Share knowledge.
- Focus on solving problems together.
- Build software that benefits everyone.

---

Expected Behavior

We encourage all collaborators to:

- Treat everyone with respect.
- Give constructive feedback.
- Write clean, understandable code.
- Document important changes.
- Test code before submitting it.
- Report bugs responsibly.
- Help other developers learn.

---

Unacceptable Behavior

The following is not allowed:

- Harassment or discrimination.
- Personal attacks or insults.
- Hate speech.
- Spam.
- Malicious code or intentional security vulnerabilities.
- Plagiarism or copyright infringement.
- Sharing confidential project information without permission.

---

Coding Standards

All collaborators should:

- Follow the project's coding style.
- Use meaningful variable and function names.
- Keep code organized.
- Write comments where necessary.
- Avoid unnecessary complexity.
- Keep commits focused and descriptive.

---

Collaboration

Before making major changes:

- Discuss the proposal with the team.
- Open an issue if needed.
- Respect code reviews.
- Be open to suggestions.

---

Security

If you discover a security vulnerability:

- Do not publish it publicly.
- Report it privately to the maintainers.
- Allow time for a fix before disclosure.

---

Enforcement

Project maintainers may:

- Request changes.
- Reject pull requests.
- Remove inappropriate content.
- Restrict or revoke collaboration privileges for repeated violations.

---

Our Mission

DreamOS is built by people who believe technology should help users achieve their goals.

Let's work together to build something extraordinary.

---

Dream. Build. Achieve.
